##  图标Icon（DYIcon）

### 相关参数

  | 参数                     | 说明             | 类型        | 可选值                              | 默认值           |
  | ------------------------ | ---------------- | ----------- | ---------------------------------- | ---------------- |
  | `name`                   | 图标名字         | `String`  | 如 `icon-test43`                         |                        |

### 使用示例
```
<template>
  <div>
   <dy-icon name="icon-test44"></dy-icon>
  </div>
</template>

<script>

export default {
  methods: {
  }
}
</script>

<style lang="less" scoped>

</style>
```
